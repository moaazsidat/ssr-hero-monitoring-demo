SSR demo for hero element monitoring
====================================

SSR react boilerplate from [Server Side Rendering with Create React App](https://github.com/ayroblu/ssr-create-react-app-v2) 

Install
-------
```bash
npm install
npm run build
npm run start:server
```
or yarn

### Development
```bash
npm start
```
or yarn

### Deploy
```bash
now
```
